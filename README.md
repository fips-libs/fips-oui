fips-oui
=========

fipsified oui-blendish (https://bitbucket.org/duangle/oui-blendish/)

fips build system: https://github.com/floooh/fips

### Building

```
> cd fips
> ./fips clone fips-oui
> cd ../fips-oui
> ./fips build
> ./fips run blendish
```
